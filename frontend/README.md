# Frontend — Crafted Charms

Customer-facing static website for the Crafted Charms jewellery store.

## Structure

```
frontend/
├── pages/         ← HTML pages (auth, product detail)
├── css/           ← Stylesheets
├── js/            ← JavaScript modules (cart, auth, products)
└── assets/
    └── images/    ← Product photos, hero images
```

## Entry Point

See `index.html` in the project root.
